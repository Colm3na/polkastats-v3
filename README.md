# Polkastats v2

> Polkadot Kusama network statistics

## Build frontend

``` bash
# Clone this repo
$ git clone https://github.com/Colm3na/polkastats.git

# Install dependencies
$ npm run install

# Serve with hot reload at localhost:3000
$ npm run dev

# Build for production
$ npm run build

# generate static project
$ npm run generate
```
